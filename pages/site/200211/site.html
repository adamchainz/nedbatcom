This web site was created with content in XML files, transformed with XSLT stylesheets. The whole process is managed by a python script directed by an XML file.

I wanted a purely text-based process that produced purely static HTML, and investigated some alternatives:

  • BaseML looked like a good alternative. I really liked their very simple scheme of documents as classes, with inheritance, and element names as methods, with inheritance. It makes for a straightforward templating architecture. In the end it didn't quite have the power I was looking for.
  • XWeb seemed like the right idea, but frankly, I couldn't figure it out.

The stellated dodecahedron (in the upper left corner) was rendered by POV-Ray, with data generated by a python script based on pypolys. There's also a larger version of it.
